# 🖨️ Λογιστικό Τυπογραφείου

Απλό, τοπικό web app για την επιχείρησή σου: έκδοση αποδείξεων/τιμολογίων, παρακολούθηση αποθήκης (στοκ), πελάτες, έξοδα και επισκόπηση εσόδων. Δεν χρειάζεται καμία συνδρομή ή εξωτερική υπηρεσία — όλα τρέχουν στον υπολογιστή σου.

## ▶️ Πώς το τρέχω

Άνοιξε ένα τερματικό (PowerShell) μέσα στον φάκελο και γράψε:

```
npm run dev
```

Μετά άνοιξε στον browser: **http://localhost:3000**

> Την πρώτη φορά μόνο, αν χρειαστεί, τρέξε `npm install` για να κατέβουν τα πακέτα.

## 📚 Τι περιλαμβάνει

| Ενότητα | Τι κάνει |
|---|---|
| **Πίνακας ελέγχου** | Τζίρος & έξοδα μήνα/έτους, κέρδος, υπόλοιπο ΦΠΑ, γράφημα, ανεξόφλητα, χαμηλό στοκ, κορυφαία προϊόντα |
| **Προσφορές** | Δημιουργία προσφορών με ισχύ, καταστάσεις (πρόχειρη/απεσταλμένη/αποδεκτή…), εκτύπωση, **επανάληψη (Reorder)**, μετατροπή σε παραγγελία ή παραστατικό |
| **Παραγγελίες** | Παρακολούθηση παραγγελιών (σε εξέλιξη/έτοιμη/παραδόθηκε), μετατροπή σε παραστατικό ή σε εργασία παραγωγής |
| **Εργασίες** | **Πίνακας παραγωγής (Kanban)**: κάθε εργασία στο στάδιό της (Σχεδιασμός → Εκτύπωση → Φινίρισμα → Έτοιμο), drag & drop, προτεραιότητες, προθεσμίες, υπεύθυνος, παραμετροποιήσιμα στάδια, ιστορικό ολοκληρωμένων |
| **Παραστατικά** | Έκδοση αποδείξεων & τιμολογίων με πολλές γραμμές, ΦΠΑ ανά είδος (προεπιλογή 19% Κύπρου), εκπτώσεις, σειρά/αρίθμηση, εκτύπωση/PDF, εισπράξεις |
| **Πελάτες** | **Καρτέλα 360°**: banner (Μικτά Έσοδα, ημέρες Χρέωσης/Πίστωσης), Τιμοκατάλογος, **Δραστηριότητες** (σημειώσεις/υπενθυμίσεις με ετικέτες), καθολικό/ledger, τα παραστατικά/προσφορές/παραγγελίες του πελάτη |
| **Προμηθευτές** | Μητρώο προμηθευτών, συνδέεται με τις Δαπάνες |
| **B2B Link** | Προσωπικό online portal ανά πελάτη (σύνδεσμος + PIN): βλέπει τον τιμοκατάλογό του, παραγγέλνει μόνος του, οι παραγγελίες έρχονται στις «Παραγγελίες» με σήμανση B2B. Ενεργοποίηση από Πελάτες → καρτέλα → B2B Link |
| **Πολυγλωσσικότητα** | Πλήρες **Αγγλικά / Ελληνικά** σε όλη την εφαρμογή (μενού, κουμπιά, φόρμες, πίνακες, μηνύματα). Εναλλαγή στιγμιαία, χωρίς reload, από τον επιλογέα **EN\|ΕΛ** στο μενού ή στις Ρυθμίσεις. Η επιλογή αποθηκεύεται και θυμάται την επόμενη φορά. |
| **Αποθήκη** | Προϊόντα/υπηρεσίες με **κατηγορίες**, αυτόματη μείωση στοκ σε κάθε πώληση, χειροκίνητες κινήσεις, ειδοποίηση χαμηλού στοκ |
| **Έξοδα** | Καταγραφή εξόδων ανά κατηγορία & προμηθευτή, με ΦΠΑ |
| **Αναφορές** | ΦΠΑ περιόδου, πωλήσεις ανά πελάτη/είδος, έξοδα ανά κατηγορία, **εξαγωγή CSV/Excel** |
| **Ρυθμίσεις** | Στοιχεία επιχείρησης (VAT Number, όχι ΑΦΜ/ΔΟΥ Ελλάδας), λογότυπο, ΦΠΑ, σειρά/αρίθμηση παραστατικών, αντίγραφο ασφαλείας |

> 🇨🇾 Η εφαρμογή είναι προσαρμοσμένη για επιχείρηση στην **Κύπρο**: προεπιλεγμένο ΦΠΑ 19%, πεδίο «VAT Number» (όχι ΑΦΜ/ΔΟΥ Ελλάδας), χωρίς myDATA/ΑΑΔΕ (αυτά είναι Ελληνικά συστήματα που δεν ισχύουν στην Κύπρο).

## 💾 Δεδομένα & αντίγραφα ασφαλείας

Όλα αποθηκεύονται τοπικά στο αρχείο **`data/db.json`**.
Από τις **Ρυθμίσεις → Αντίγραφο ασφαλείας** μπορείς να κατεβάζεις όποτε θέλεις ένα πλήρες αντίγραφο. Κράτα τακτικά αντίγραφα (π.χ. σε USB ή cloud).

## 🧾 Ροή εργασίας (τυπική μέρα)

1. **Ρυθμίσεις**: συμπλήρωσε μία φορά τα στοιχεία της επιχείρησης, το λογότυπο και το ΦΠΑ.
2. **Αποθήκη**: καταχώρησε τα προϊόντα/υπηρεσίες σου με τιμές.
3. **Πελάτες**: πρόσθεσε πελάτες (χρειάζεται κυρίως για τιμολόγια).
4. **Παραστατικά → Νέο**: διάλεξε είδη, ποσότητες → «Έκδοση» → «Εκτύπωση/PDF».
   Το στοκ μειώνεται αυτόματα.
5. **Έξοδα**: κατέγραψε αγορές/λειτουργικά.
6. **Πίνακας ελέγχου**: δες πώς πάει ο μήνας.
